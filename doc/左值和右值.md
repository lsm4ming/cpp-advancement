### 定义

1. 出现等号左边的表达式是左值；
2. 一个变量不是左值即是右值；
3. 一个变量不可以同时即是左值也是右值；
4. 左值某些情况下可以是右值；

````cpp
int a = 10; // a 是左值，10 是右值
a = a + 1; // a 被当成右值使用
````

### 为什么需要右值

````
主要是为了实现移动语义，以及避免不必要的复制。
右值引用（&&）允许 C++ 实现更高效的资源管理，使用 std::move 将一个对象的资源转移到另一个对象时，右值引用可以帮助避免昂贵的深拷贝操作，提升程序性能
````

### i++和++i

````cpp
int a = 1;
int &b = ++a; // 左值
int &&c = a++; // 右值
````

* ++i：左值
````
++i等价于i=i+1，返回的变量是i本身
````
* i++：右值
````
i++的过程产生了临时变量，返回的变量是临时变量本身
````
